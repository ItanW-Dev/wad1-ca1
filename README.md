Title - Ibanez Guitar Collection

This Website is made to showcase a few of Ibanez's guitar series
You start on the sign up and login page, once you create an account and login you get to the start page
The Dashboard page lets you click into each series and view the guitars in them, you can add, remove, edit, guitars and you can make guitar collections with different pictures.
The about page gives you a bit of information and the location of Ibanez offices.
The stats page lets you see a few stats overall on the website and some only for the user.
